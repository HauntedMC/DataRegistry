# Support

## Where to Get Help

- Setup/configuration help: https://www.hauntedmc.nl/support
- Bug reports: GitHub Issues (use issue templates)
- Feature proposals: GitHub Issues (feature request template)
- Security concerns: [SECURITY.md](SECURITY.md)

Please keep support questions in the channels above so context is visible to maintainers.

## Before Opening an Issue

Please provide:

- DataRegistry version
- DataProvider version
- Platform and version (Velocity / Paper)
- Java version
- Enabled feature flags
- Relevant config section(s)
- Logs / stack traces
- Clear reproduction steps

## Response Notes

Support and triage are best-effort and prioritized by impact + reproducibility.
